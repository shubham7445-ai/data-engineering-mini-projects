# Data Engineering Mini Projects

This repository contains two simple data-engineering style projects suitable for an internship portfolio:

1. **Sales Analytics Dashboard (Sales Analytics)** – SQL-based data model for sales, customers, and products.
2. **Student Data Pipeline (Mini ETL)** – Python + SQL mini ETL pipeline for student records.

Both projects are designed to demonstrate:
- Basic SQL skills (tables, joins, aggregations)
- Simple data modelling
- Basic Python for ETL-style tasks
